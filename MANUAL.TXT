### 4. **concept_extractor.py** (434 linhas)
**O que faz:** Usa LLM para extrair **conceitos pedagógicos** do livro.
**O problema que resolve:** Regex não consegue entender o que é "conceito". Só o LLM consegue dizer que "vetor" é um conceito e "exemplo de vetor" não é.
**Estratégia:**
1. Divide livro em chunks de 2500 palavras
2. Envia cada chunk ao LLM
3. LLM retorna lista de conceitos com:
   - `id`: slug único (ex: `variavel_inteira`)
   - `nome`: nome legível
   - `complexidade`: baixa/media/alta/critica
   - `dependencias`: IDs de conceitos que precisa entender antes
   - `keywords`: palavras-chave
4. Deduplica conceitos similares
5. Valida dependências (remove ciclos)

5. decisor.py (206 linhas)
O que faz: Calcula matematicamente quantos episódios são necessários.
O problema que resolve: O LLM inventava números ("3 episódios" ou "12 episódios") sem base real.
A fórmula:
score = segmentos_base[complexidade]
      + (4 se tem_codigo)
      + (3 se tem_formula)
      + (subconcepts × 2)
      + (dependencies × 1)
segmentos_min = { baixa: 8, media: 12, alta: 18, critica: 22 }
total_episodios = max(
    total_segmentos / 40,  # por capacidade
    len(conceitos) / 3     # por foco (max 3 conceitos/ep)
)
Resultado:
ResultadoDecisao:
├── total_episodios: int
├── total_segmentos_necessarios: int
├── conceitos_rankeados: list[Conceito]  # ordenados por score
└── detalhes_por_conceito: list[dict]    # debug

6. grouper.py (377 linhas)
O que faz: Agrupa conceitos em episódios respeitando dependências.
Algoritmo - Ordenação Topológica (Kahn's):
1. Calcula grau de entrada de cada conceito
2. Conceitos sem deps vão primeiro
3. Processa fila: adiciona resultado, decrementa deps dos dependentes
4. Conceitos que chegam a grau 0 entram na fila
Resultado:
PlanoCompleto:
├── total_episodios: int
├── episodios: list[EpisodioPlano]
│   ├── numero: int
│   ├── titulo_sugerido: str
│   ├── conceitos: list[str]  # IDs dos conceitos
│   ├── depth_level: DepthLevel
│   ├── chunk_texto: str  # texto do livro para este ep
│   └── conceitos_preparar: list[str]  # para plantar semente
└── cobertura_percentual: float

### 7. **coverage_check.py** (290 linhas)
**O que faz:** **Validação de 100% de cobertura** antes de gerar roteiros.
**Verificações críticas:**
1. ✅ Todo conceito está em exatamente 1 episódio (sem lacuna, sem duplicata)
2. ✅ Nenhum episódio vazio
3. ✅ Ordem de dependências respeitada (B não vem antes de A se B depende de A)
4. ✅ Nenhum episódio com >3 conceitos
5. ✅ Todos os chunks têm >50 palavras
6. ✅ Estimativa de segmentos >=20 por episódio
**Se falhar:** O pipeline **PARA** e diz exatamente o que está errado.

"""
Gerador de Áudio para Podcasts - FABOT Planner
Usa Edge TTS com vozes definidas no fabot-studio
"""
import asyncio
import re
import subprocess
from pathlib import Path
import edge_tts
from pydub import AudioSegment
VOICES = {
    "NARRADOR": {
        "voice": "pt-BR-ThalitaMultilingualNeural",
        "rate": "-5%",
        "pitch": "+0Hz",
    },
    "WILLIAM": {
        "voice": "pt-BR-AntonioNeural",
        "rate": "+10%",
        "pitch": "+0Hz",
    },
    "CRISTINA": {
        "voice": "pt-BR-AntonioNeural",
        "rate": "+10%",
        "pitch": "+0Hz",
    },
}
PAUSES = {
    "same_speaker": 300,
    "between_speakers": 550,
    "after_narrador": 1800,
    "block_transition": 1200,
    "end_of_episode": 2000,
}
def parse_roteiro(texto: str) -> list[dict]:
    """Parse do arquivo TXT para lista de segmentos."""
    segments = []
    lines = texto.strip().split('\n')
    
    for line in lines:
        line = line.strip()
        if not line:
            continue
        
        # Parse [ 1] [NARRADOR]
        match = re.match(r'\[\s*\d+\]\s*\[(\w+)\]', line)
        if match:
            speaker = match.group(1).upper()
            
            # Extrair o texto após o nome do speaker
            text_start = line.find(']', match.start(1)) + 1
            text = line[text_start:].strip()
            
            if text and len(text) > 3:
                segments.append({
                    "speaker": speaker,
                    "text": text,
                    "pause_after_ms": PAUSES.get("between_speakers", 550),
                    "block_transition": False,
                })
    
    return segments
async def synthesize_segment(text: str, voice_config: dict, output_path: Path) -> Path:
    """Gera MP3 de um segmento."""
    communicate = edge_tts.Communicate(
        text,
        voice_config["voice"],
        rate=voice_config["rate"],
        pitch=voice_config["pitch"]
    )
    
    audio_data = b""
    async for chunk in communicate.stream():
        if chunk["type"] == "audio":
            audio_data += chunk["data"]
    
    output_path.write_bytes(audio_data)
    return output_path
async def generate_audio(segments: list, output_file: str, title: str):
    """Gera o áudio completo do podcast."""
    print(f"\n{'='*60}")
    print(f"Gerando: {title}")
    print(f"Segmentos: {len(segments)}")
    print(f"{'='*60}")
    
    output_path = Path(output_file)
    output_path.parent.mkdir(parents=True, exist_ok=True)
    
    temp_dir = output_path.parent / "temp_segments"
    temp_dir.mkdir(exist_ok=True)
    
    # Gerar segmentos em paralelo
    semaphore = asyncio.Semaphore(3)
    
    async def synth_seg(i, seg):
        async with semaphore:
            voice_key = seg["speaker"]
            if voice_key not in VOICES:
                voice_key = "WILLIAM"  # fallback
                
            voice_config = VOICES[voice_key]
            seg_path = temp_dir / f"seg_{i:03d}.mp3"
            
            print(f"  [{i+1}/{len(segments)}] {seg['speaker']}: {seg['text'][:50]}...")
            
            await synthesize_segment(seg["text"], voice_config, seg_path)
            
            return {
                "path": seg_path,
                "speaker": seg["speaker"],
                "pause_after_ms": seg.get("pause_after_ms", 550),
                "block_transition": seg.get("block_transition", False),
            }
    
    tasks = [synth_seg(i, seg) for i, seg in enumerate(segments)]
    results = await asyncio.gather(*tasks)
    
    # Montar áudio final
    print("\nMontando áudio...")
    episode = AudioSegment.silent(duration=1500)
    
    for i, seg in enumerate(results):
        if seg is None or not seg["path"].exists():
            continue
            
        audio = AudioSegment.from_mp3(str(seg["path"]))
        audio = audio.fade_in(30)
        
        # APLICAR AUMENTO DE VOLUME (+5dB para som mais alto)
        audio = audio + 5  # Aumenta volume em 5dB
        
        episode += audio
        
        # Calcular pausa
        if seg["block_transition"]:
            pause_ms = PAUSES["block_transition"]
        elif seg["speaker"] == "NARRADOR":
            pause_ms = PAUSES["after_narrador"]
        elif i + 1 < len(results) and results[i + 1]:
            if results[i + 1]["speaker"] == seg["speaker"]:
                pause_ms = PAUSES["same_speaker"]
            else:
                pause_ms = PAUSES["between_speakers"]
        else:
            pause_ms = PAUSES["end_of_episode"]
        
        episode += AudioSegment.silent(duration=pause_ms)
    
    episode += AudioSegment.silent(duration=2000)
    
    # Normalizar para volume consistente
    print("Normalizando volume...")
    change_in_dB = -episode.max_dBFS + (-16)
    episode = episode.apply_gain(change_in_dB)
    
    # Aumentar volume final (+3dB extra para ficar mais alto)
    episode = episode + 3
    
    # Exportar
    print(f"Exportando para {output_file}...")
    episode.export(
        str(output_path),
        format="mp3",
        bitrate="192k",
        parameters=["-ar", "44100", "-ac", "2"],
    )
    
    # Limpar temp
    for f in temp_dir.glob("*.mp3"):
        f.unlink()
    temp_dir.rmdir()
    
    duracao = len(episode) / 1000
    tamanho = output_path.stat().st_size / 1024 / 1024
    
    print(f"✅ Áudio gerado: {duracao:.0f}s ({duracao/60:.1f} min) | {tamanho:.1f} MB")
    
    return output_path

        audio = AudioSegment.from_mp3(str(seg["path"]))
        audio = audio.fade_in(30)
        
        # APLICAR AUMENTO DE VOLUME (+5dB para som mais alto)
        audio = audio + 5  # Aumenta volume em 5dB
        
        episode += audio
        
        # Calcular pausa
        if seg["block_transition"]:
            pause_ms = PAUSES["block_transition"]
        elif seg["speaker"] == "NARRADOR":
            pause_ms = PAUSES["after_narrador"]
        elif i + 1 < len(results) and results[i + 1]:
            if results[i + 1]["speaker"] == seg["speaker"]:
                pause_ms = PAUSES["same_speaker"]
            else:
                pause_ms = PAUSES["between_speakers"]
        else:
            pause_ms = PAUSES["end_of_episode"]
        
        episode += AudioSegment.silent(duration=pause_ms)
    
    episode += AudioSegment.silent(duration=2000)
    
    # Normalizar para volume consistente
    print("Normalizando volume...")
    change_in_dB = -episode.max_dBFS + (-16)
    episode = episode.apply_gain(change_in_dB)
    
    # Aumentar volume final (+3dB extra para ficar mais alto)
    episode = episode + 3
    
    # Exportar
    print(f"Exportando para {output_file}...")
    episode.export(
        str(output_path),
        format="mp3",
        bitrate="192k",
        parameters=["-ar", "44100", "-ac", "2"],
    )
    
    # Limpar temp
    for f in temp_dir.glob("*.mp3"):
        f.unlink()
    temp_dir.rmdir()
    
    duracao = len(episode) / 1000
    tamanho = output_path.stat().st_size / 1024 / 1024
    
    print(f"✅ Áudio gerado: {duracao:.0f}s ({duracao/60:.1f} min) | {tamanho:.1f} MB")
    
    return output_path

"""

from youtube_transcript_api import YouTubeTranscriptApi
from youtube_transcript_api._errors import (

def get_video_id(url: str) -> str:
    """Extrai o ID do vídeo da URL."""
    if "youtu.be" in url:
        return url.split("/")[-1].split("?")[0]
    elif "watch?v=" in url:
        return url.split("watch?v=")[-1].split("&")[0]
    else:
        return url.split("/")[-1].split("?")[0]
def transcrever_video(video_url: str, idioma_original: str = "pt", idioma_fallback: str = "en") -> dict:
    """
    Transcreve um vídeo do YouTube.
    
    Returns:
        dict com:
            - video_id: str
            - titulo: str (se disponível)
            - transcricao: list de {text, start, duration}
            - texto_completo: str (todo texto concatenado)
            - idioma: str
            - sucesso: bool
            - erro: str (se falhou)
    """
    video_id = get_video_id(video_url)
    
    print(f"📺 Video ID: {video_id}")
    print(f"🌐 Buscando transcrição em {idioma_original}...")
    
    try:
        # Tenta pegar a transcrição
        transcript = YouTubeTranscriptApi.get_transcript(
            video_id,
            languages=[idioma_original, idioma_fallback, 'pt-BR', 'en-US']
        )
        
        # Processar transcrição
        segmentos = []
        texto_completo = []
        
        for item in transcript:
            segmentos.append({
                "texto": item['text'],
                "inicio": item['start'],
                "duracao": item['duration'],
                "fim": item['start'] + item['duration']
            })
            texto_completo.append(item['text'])
        
        # Detectar idioma da transcrição
        idioma_detectado = idioma_original
        
        # Calcular duração total
        duracao_total = max(item['start'] + item['duration'] for item in transcript) if transcript else 0
        
        resultado = {
            "video_id": video_id,
            "video_url": video_url,
            "transcricao": segmentos,
            "texto_completo": " ".join(texto_completo),
            "idioma": idioma_detectado,
            "idioma_original": idioma_original,
            "num_segmentos": len(segmentos),
            "duracao_segundos": duracao_total,
            "duracao_minutos": round(duracao_total / 60, 1),
            "num_palavras": len(" ".join(texto_completo).split()),
            "sucesso": True,
            "erro": None,
            "data_transcricao": datetime.now().isoformat()
        }
        
        print(f"✅ Transcrição completa!")
        print(f"   Duração: {resultado['duracao_minutos']} minutos")
        print(f"   Segmentos: {resultado['num_segmentos']}")
        print(f"   Palavras: {resultado['num_palavras']}")
        
        return resultado
        
    except TranscriptsDisabled:
        return _erro(video_id, video_url, "Transcrições desabilitadas para este vídeo")
    
    except NoTranscriptFound:
        return _erro(video_id, video_url, "Nenhuma transcrição encontrada neste vídeo")
    
    except VideoUnavailable:
        return _erro(video_id, video_url, "Vídeo indisponível ou privado")
    
    except Exception as e:
        return _erro(video_id, video_url, f"Erro inesperado: {str(e)}")
def _erro(video_id: str, url: str, msg: str):
    """Retorna estrutura de erro."""
    print(f"❌ Erro: {msg}")
    return {
        "video_id": video_id,
        "video_url": url,
        "transcricao": [],
        "texto_completo": "",
        "idioma": None,
        "num_segmentos": 0,
        "duracao_segundos": 0,
        "duracao_minutos": 0,
        "num_palavras": 0,
        "sucesso": False,
        "erro": msg,
        "data_transcricao": datetime.now().isoformat()
    }
def salvar_transcricao(resultado: dict, output_path: str) -> str:
    """Salva a transcrição em JSON e TXT."""
    output = Path(output_path)
    output.parent.mkdir(parents=True, exist_ok=True)
    
    # Salvar JSON completo
    json_path = output.with_suffix('.json')
    with open(json_path, 'w', encoding='utf-8') as f:
        json.dump(resultado, f, ensure_ascii=False, indent=2)
    
    # Salvar TXT legível
    txt_path = output.with_suffix('.txt')
    linhas = [
        "=" * 70,
        f"TRANSCRIÇÃO DE VÍDEO",
        "=" * 70,
        f"Video ID: {resultado['video_id']}",
        f"URL: {resultado['video_url']}",
        f"Idioma: {resultado['idioma']}",
        f"Duração: {resultado['duracao_minutos']} minutos",
        f"Palavras: {resultado['num_palavras']}",
        f"Data: {resultado['data_transcricao']}",
        "",
        "=" * 70,
        "TEXTO COMPLETO",
        "=" * 70,
        "",
        resultado['texto_completo'],
        "",
        "=" * 70,
        "SEGMENTOS (com timestamps)",
        "=" * 70,
    ]
    
    for seg in resultado.get('transcricao', []):
        minutos = int(seg['inicio'] // 60)
        segundos = int(seg['inicio'] % 60)
        linhas.append(f"[{minutos:02d}:{segundos:02d}] {seg['texto']}")
    
    with open(txt_path, 'w', encoding='utf-8') as f:
        f.write("\n".join(linhas))
    
    print(f"💾 Salvo em:")
    print(f"   JSON: {json_path}")
    print(f"   TXT: {txt_path}")
    
    return str(txt_path)

"""
Transcritor de YouTube para FABOT Podcast
Transcreve vídeos e salva para uso no pipeline
"""
from youtube_transcript_api import YouTubeTranscriptApi
from youtube_transcript_api._errors import (
    TranscriptsDisabled,
    NoTranscriptFound,
    VideoUnavailable,
)
import json
from pathlib import Path
from datetime import datetime
def get_video_id(url: str) -> str:
    """Extrai o ID do vídeo da URL."""
    if "youtu.be" in url:
        return url.split("/")[-1].split("?")[0]
    elif "watch?v=" in url:
        return url.split("watch?v=")[-1].split("&")[0]
    else:
        return url.split("/")[-1].split("?")[0]
def transcrever_video(
    video_url: str, idioma_original: str = "pt", idioma_fallback: str = "en"
) -> dict:
    """
    Transcreve um vídeo do YouTube.
    
    Returns:
        dict com:
            - video_id: str
            - titulo: str (se disponível)
            - transcricao: list de {text, start, duration}
            - texto_completo: str (todo texto concatenado)
            - idioma: str
            - sucesso: bool
            - erro: str (se falhou)
    """
    video_id = get_video_id(video_url)
    print(f"📺 Video ID: {video_id}")
    print(f"🌐 Buscando transcrição em {idioma_original}...")
    try:
        # Tenta pegar a transcrição usando a nova API
        api = YouTubeTranscriptApi()
        transcript = api.fetch(
            video_id=video_id,
            languages=[idioma_original, idioma_fallback, "pt-BR", "en-US"],
        )
        
        # O fetch() retorna um FetchedTranscript com .snippets
        snippets = list(transcript.snippets)
        if not snippets:
            return _erro(video_id, video_url, "Nenhuma transcrição disponível")
        
        # Processar transcrição
        segmentos = []
        texto_completo = []
        for item in snippets:
            segmentos.append({
                "texto": item.text,
                "inicio": item.start,
                "duracao": item.duration,
                "fim": item.start + item.duration,
            })
            texto_completo.append(item.text)
        # Detectar idioma da transcrição
        idioma_detectado = transcript.language
        # Calcular duração total
        duracao_total = max(item.start + item.duration for item in snippets) if snippets else 0
        resultado = {
            "video_id": video_id,
            "video_url": video_url,
            "transcricao": segmentos,
            "texto_completo": " ".join(texto_completo),
            "idioma": idioma_detectado,
            "idioma_original": idioma_original,
            "num_segmentos": len(segmentos),
            "duracao_segundos": duracao_total,
            "duracao_minutos": round(duracao_total / 60, 1),
            "num_palavras": len(" ".join(texto_completo).split()),
            "sucesso": True,
            "erro": None,
            "data_transcricao": datetime.now().isoformat(),
        }
        print(f"✅ Transcrição completa!")
        print(f"   Duração: {resultado['duracao_minutos']} minutos")
        print(f"   Segmentos: {resultado['num_segmentos']}")
        print(f"   Palavras: {resultado['num_palavras']}")
        return resultado
    except TranscriptsDisabled:
        return _erro(video_id, video_url, "Transcrições desabilitadas para este vídeo")
    except NoTranscriptFound:
        return _erro(video_id, video_url, "Nenhuma transcrição encontrada neste vídeo")
    except VideoUnavailable:
        return _erro(video_id, video_url, "Vídeo indisponível ou privado")
    except Exception as e:
        return _erro(video_id, video_url, f"Erro inesperado: {str(e)}")
def _erro(video_id: str, url: str, msg: str):
    """Retorna estrutura de erro."""
    print(f"❌ Erro: {msg}")
    return {
        "video_id": video_id,
        "video_url": url,
        "transcricao": [],
        "texto_completo": "",
        "idioma": None,
        "num_segmentos": 0,
        "duracao_segundos": 0,
        "duracao_minutos": 0,
        "num_palavras": 0,
        "sucesso": False,
        "erro": msg,
        "data_transcricao": datetime.now().isoformat(),
    }
def salvar_transcricao(resultado: dict, output_path: str) -> str:
    """Salva a transcrição em JSON e TXT."""
    output = Path(output_path)
    output.parent.mkdir(parents=True, exist_ok=True)
    # Salvar JSON completo
    json_path = output.with_suffix(".json")
    with open(json_path, "w", encoding="utf-8") as f:
        json.dump(resultado, f, ensure_ascii=False, indent=2)
    # Salvar TXT legível
    txt_path = output.with_suffix(".txt")
    linhas = [
        "=" * 70,
        f"TRANSCRIÇÃO DE VÍDEO",
        "=" * 70,
        f"Video ID: {resultado['video_id']}",
        f"URL: {resultado['video_url']}",
        f"Idioma: {resultado['idioma']}",
        f"Duração: {resultado['duracao_minutos']} minutos",
        f"Palavras: {resultado['num_palavras']}",
        f"Data: {resultado['data_transcricao']}",
        "",
        "=" * 70,
        "TEXTO COMPLETO",
        "=" * 70,
        "",
        resultado["texto_completo"],
        "",
        "=" * 70,
        "SEGMENTOS (com timestamps)",
        "=" * 70,
    ]
    for seg in resultado.get("transcricao", []):
        minutos = int(seg["inicio"] // 60)
        segundos = int(seg["inicio"] % 60)
        linhas.append(f"[{minutos:02d}:{segundos:02d}] {seg['texto']}")
    with open(txt_path, "w", encoding="utf-8") as f:
        f.write("\n".join(linhas))
    print(f"💾 Salvo em:")
    print(f"   JSON: {json_path}")
    print(f"   TXT: {txt_path}")
    return str(txt_path)

async def gerar_audio_episodio(episodio_json_path: str, output_mp3_path: str):
    """Gera áudio de um episódio JSON."""
    with open(episodio_json_path, 'r', encoding='utf-8') as f:
        data = json.load(f)
    
    # Parse do JSON gerado pelo planner
    segments = []
    for seg in data.get('segments', []):
        segments.append({
            "speaker": seg.get('speaker', 'WILLIAM').upper(),
            "text": seg.get('text', ''),
            "pause_after_ms": seg.get('pause_after_ms', 550),
            "block_transition": seg.get('block_transition', False),
        })
    
    if not segments:
        print(f"⚠️ Nenhum segmento encontrado em {episodio_json_path}")
        return None
    
    title = data.get('title', 'Episódio')
    return await generate_audio(segments, output_mp3_path, title)
